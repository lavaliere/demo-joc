Hier wird ausgewählt, wie das Build aus dem Artefakte kopiert werden sollen, selektiert wird. Zum Beispiel das letzte erfolgreiche und stabile Build oder das letzte "Keep Forever" Build.
Andere Plugins können weitere Auswahlmöglichkeiten hinzufügen.
Die Buildnummer des ausgewählten Builds wird in einer Umgebungsvariable als Referenz für weitere Buildschritte gespeichert. Der Name der Umgebungsvariable ist COPYARTIFACT_BUILD_NUMBER_ mit dem Namen des Quellprojekts angehängt (alles in Großbuchstaben, alle Zeichen außer A-Z werden durch Unterstriche ersetzt, wobei statt mehrerer aufeinanderfolgender Unterstriche nur ein einziger verwendet wird).

Beispiel:

  • Die Umgebungsvariable COPYARTIFACT_BUILD_NUMBER_PROJECT_ABC enthält die Buildnummer des aus dem Projekt Project-ABC kopierten Builds.
  • Die Umgebungsvariable COPYARTIFACT_BUILD_NUMBER_TOOL_RELEASE_ enthält die Buildnummer des aus dem Projekt tool1-release1.2 kopierten Builds.
